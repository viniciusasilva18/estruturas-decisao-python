# 📘 Exercícios de Estruturas de Decisão em Python

Esse repositório reúne exercícios que fiz para treinar Python, principalmente usando estruturas de decisão (if, elif e else).

Aqui você vai encontrar problemas simples do dia a dia, como cálculos, validações e menus, tudo organizado em arquivos separados pra facilitar o estudo.

💡 A ideia foi praticar lógica de programação e melhorar meu raciocínio na resolução de problemas.

🚀 Projeto ideal pra quem está começando em Python ou quer reforçar a base.
